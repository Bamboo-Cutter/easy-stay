# Easy Stay Mobile (Taro)

移动端预订流程项目，基于 **Taro + React + TypeScript**。

## 已完成页面

- 首页：`pages/home`
- 酒店列表：`pages/hotel-list`
- 酒店详情：`pages/hotel-detail`
- 房型报价：`pages/room-offers`
- 预订页：`pages/booking`

## 运行

```bash
cd /Users/mac/Desktop/easy-stay-hotel/apps/mobile
pnpm install
pnpm dev:h5
```

微信小程序：

```bash
pnpm dev:weapp
```

## 后端 API

默认调用：`http://localhost:3000`

请先启动后端服务。
